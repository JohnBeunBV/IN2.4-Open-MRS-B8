# Grafana Dashboard JSON bestanden

Download de volgende dashboards van https://grafana.com/grafana/dashboards
en sla ze op in deze map met de aangegeven bestandsnaam.

| Bestandsnaam        | Dashboard ID | Beschrijving                        |
|---------------------|--------------|-------------------------------------|
| spring-boot.json    | 17175        | Spring Boot 3.x - HTTP metrics      |
| jvm.json            | 4701         | JVM Micrometer - heap, GC, threads  |
| rabbitmq.json       | 10991        | RabbitMQ - queues, message rates    |

## Hoe downloaden?
1. Ga naar https://grafana.com/grafana/dashboards/<ID>
2. Klik op "Download JSON"
3. Sla op in deze map met de naam uit de tabel hierboven
